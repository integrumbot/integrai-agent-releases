# IntegrAI-Agent — Агент для удалённой печати

Печатайте документы на любом принтере прямо из Telegram — из любой точки мира.

## Скачать

**[Скачать последнюю версию](https://github.com/integrumbot/integrai-print-releases/releases/latest)**

| Платформа | Файл | Размер |
|-----------|------|--------|
| Windows 10/11 | `IntegrAI-PRINT.exe` | ~72 MB |

## Как начать

1. Откройте Telegram-бота [@IntegrAI_PRINT_bot](https://t.me/IntegrAI_PRINT_bot)
2. Зарегистрируйтесь и создайте принтер
3. Скачайте и установите агент на компьютер с принтером
4. Привяжите агент к принтеру по коду из бота
5. Отправьте файл боту — он напечатается автоматически

## Поддерживаемые форматы

PDF, DOC, DOCX, XLSX, TXT, JPG, PNG, BMP, GIF, TIFF
